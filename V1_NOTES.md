# Gateway Pulse v0.9 / v1.0 Direction

This build simplifies the operator dashboard.

Main dashboard:
- Relay running/stopped
- Trimode running/offline
- Scanner scanning/stopped
- Command port OK/fail
- Current frequency
- SFI
- Last station/session
- Recent activity

No health score.
No gimmick gauges.
No auto-control.
No auto-update.

Advanced diagnostics are hidden under a collapsible section.
