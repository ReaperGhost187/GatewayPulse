Gateway Pulse is a read-only monitoring dashboard
for RMS Relay and RMS Trimode gateways.

It NEVER controls Winlink.
It NEVER transmits.
It NEVER modifies Relay or Trimode.

--------------------------------------------

Dashboard

Open your browser and go to:

http://127.0.0.1:8080

To view from another device on your LAN:

http://YOUR_GATEWAY_IP:8080

Example:

http://192.168.1.100:8080

--------------------------------------------

Pushover Notifications (Optional)

Edit:

appsettings.json

Set:

"Enabled": true

Then enter your:

UserKey
ApiToken

Restart the Gateway Pulse service.

--------------------------------------------

Requirements

Windows 11
RMS Relay
RMS Trimode
